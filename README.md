# 🏴‍☠️ Pirate vs Normal Chatbot — LLaMA 3.1 LLM App

An interactive dual-personality chatbot powered by **Meta LLaMA 3.1**, built with a sleek **Gradio interface**.

Switch instantly between:

🤖 **Professional AI Assistant Mode**
🏴‍☠️ **Pirate Roleplay Mode (fun, stylized responses)**

This project demonstrates how modern LLMs can be controlled through **prompt engineering + UI design** to create distinct conversational personas.

---

## ✨ Project Highlights

🔥 Real-time LLM responses with streaming
🧠 Role-based AI behavior via prompts
🎛️ Interactive UI with mode switching
⚡ Runs on Google Colab (GPU-friendly)
🌐 Hugging Face Hub integration
🎨 Clean, user-friendly interface

---

## 🧠 How It Works

1. User selects a mode (Normal or Pirate)
2. Prompt template adjusts model behavior
3. Meta LLaMA 3.1 generates responses
4. Gradio streams output live to the UI

➡️ Same model, completely different personality.

## 📦 Installation & Setup

### 🔹 Clone Repository

```bash
git clone https://github.com/Tanmay1112004/pirate-vs-normal-chatbot.git
cd pirate-vs-normal-chatbot
```

---

### 🔹 Install Dependencies

```bash
pip install -r requirements.txt
```

**requirements.txt**

```
transformers
accelerate
torch
gradio
```

---

### 🔹 Run on Google Colab (Recommended)

1. Open `notebook.ipynb` in Colab
2. Add your Hugging Face access token
3. Run all cells 🚀
